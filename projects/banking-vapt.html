Banking VAPT — Web & API

Scope: Web application + API endpoints (authentication, payments, user mgmt)

Summary

Performed grey-box testing with credentials, identified critical issues including stored XSS and IDOR in customer payout flows. Produced PoCs and step-by-step remediation guidance.

Key Findings

  • Stored XSS in user messages — PoC: payload that executes on admin console (risk: account takeover).
  • IDOR on transaction approval — allowed access to other customers' transactions.
  • JWT misconfiguration — tokens allowed algorithm confusion in older libraries.

Remediation

  1. Sanitize and encode user inputs; use CSP for admin interfaces.
  2. Implement authorization checks server-side for transaction resources.
  3. Enforce strict JWT algorithms and rotate keys, add revocation list.
