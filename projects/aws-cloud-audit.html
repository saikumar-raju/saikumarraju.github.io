AWS Cloud Audit — Hardening & IAM

Scope: IAM policies, S3, Security Groups, CloudTrail, VPC.

Summary

Found overly-permissive IAM roles and public S3 buckets. Recommended least-privilege policies, S3 encryption and CloudTrail log centralization.

Key Actions

  • Converted broad IAM roles to scoped roles with resource-level permissions.
  • Locked down S3 buckets, enabled encryption & MFA delete policy.
  • Implemented centralized CloudTrail and log monitoring alerts.
